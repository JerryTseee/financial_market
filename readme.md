# Intro
## functions of financial market
世界上最早的证券交易所 - 荷兰阿姆斯特丹交易所, 1773年成立伦敦交易所, 1790年美国成立第一个交易所费城证券交易所, 1817年成立纽约证券交易所
- 资金流通 (资金在供给者和需求者之间转移)
- 价格发现 (供求双方的相互作用决定了交易资产的价格)
- 提供流动性 (金融资产交易变现或赎回的渠道和机制)
- 风险管理 (通过保险, 对冲交易等方式对金融行为的风险进行管理)
- 降低搜寻成本和信息成本 (通过金融市场寻找)
## financing (融资)
- 直接融资: 没有经过第三方, 通过直接与资金需求单位协议 (股票市场融资, 债券市场融资, 风险投资融资, 商业信用融资, 民间借贷)
- 间接融资: 经过第三方, 供给者和需求者没有直接接触, 人们通过存款等形式购买银行, 信托, 保险等机构发行的有价证券, 金融机构再以贷款等方式把资金给需求者
## different financial markets
- 货币市场(一年以内, 短期国库券市场, 商业票据市场)
- 资本市场(一年以上, 股票市场, 中长期国债市场, 中长期银行贷款市场)
- 一级市场(发行市场， 用于筹集资金)
- 二级市场(流通市场, 提供流通性)
- 交易所市场(股票交易所)
- 场外交易市场(黄金市场, ...)
## financial companies
- 存款吸收机构(eg.商业银行)
- 中央银行(控制金融体系的关键)
- 其他金融公司(保险公司, 财务公司, 证券交易商, 投资基金)
"银行：中央银行，政策性银行，商业银行，投资银行，世界银行，它们的职责各不相同"

# China Financial Markets
## Intro
中央银行为主导, 国有银行为主体；拥有以证券公司，期货公司和证券投资基金为主，以各类投资咨询中介，信托机构为辅的多元化投资中介体系；并拥有多种保险服务的保险中介体系
- 商业银行业务（负债业务，资产业务，表外业务, ...）
- 证券公司业务（证券经纪业务，证券投资咨询业务，交易投资活动, ...）
- 保险公司业务（家庭/企业保险，机动车保险，健康/意外保险，责任保险，特殊风险保险，信用保证保险，投资型保险，人身保险，分红型保险，...）
## Government
国务院金融稳定发展委员会职责在审议金融业，落实中央决策，统筹发展，...
中国人民银行负责制定和执行货币政策，维护金融稳定，承担一定金融职务
## Monetary Policy（货币政策）
中央银行利用控制货币供给量，来达到影响其他经济活动所采取的措施，达到稳定物价，充分就业，经济增长，金融稳定
货币供应量上升 -> 实际利率下降 -> 投资上升 -> 总产出上升
货币供应量上升 -> 银行存款和贷款上升 -> 投资上升 -> 总产出上升
托宾Q理论：Q值定义为企业市场价值与资本重置成本之比，若Q值大于1，企业市价大于重置成本，投资增加，提高社会总需求和总产出
货币供应量上升 -> 股价上升 -> 托宾Q值上升 -> 投资上升 -> 总产出上升

# 股票交易
- 指令驱动（订单驱动市场，证券价格由市场上买方订单和卖方订单共同驱动）

- 融资融券交易（“借钱买证券”）
- 杠杆交易（利用小额资金来进行数倍于原始金额的投资，风险大但收益高）

## 基本面分析法（长期）
股票分析师根据经济学，金融学，财务管理学，投资学等原理来对股票价值及其价格基本要素进行分析，判断其投资价值和合理价位
## 技术分析法（短期）
仅从股票的市场行为来分析股票价格未来变化趋势，例如通过市场价格，成交量，变化量，时间等
## 量化分析法
利用统计，数值模拟模型等进行证券市场研究的一种方法。使用大量数据，模型和电脑，应用于证券估值，组合构造优化，策略制定，风险管理等

# 股票绝对估值法
假设价值来源于未来流入的现金流，将这一系列现金流变成present value，便得到相应价值
# 股票相对估值法
可以比较其他公司的价格为基础，来评估目标公司的相应价值


# 债券（bond）"风险小于股票"
是一种有价证券，是社会各类经纪主体为了筹集资金而向债券投资者出具的，承诺按一定利率定期支付利息并到期偿还本金的债权债务凭证
- 票面价值
- 到期期限（短期，中期，长期）
- 票面利率（固定利率，浮动利率，可调利率）
- 发行者名称（企业公司，中央政府，地方政府，...）
- 公募债券 vs 私募债券

# 债券 vs 股票
- 它们都是有价证券
- 都是直接融资工具
- 债券获取资金属于公司负债，股票筹集资金列入公司资本
- 债券有很多经济体可以发行，股票只有上市公司才可发行
- 债券有期限，股票没有期限
- 债券有固定利息，股票没有固定股息

# 债券交易
- 现券交易（证券买卖双方在成交后就办理交收手续，二级市场交易）
- 回购交易（债券买卖双方在成交同时，约定于未来某一时间再次反向交易的行为，投资者在价格差中得利）
- 远期交易（双方约定在未来某一时间约定按照现在价格来交易）
- 期货交易（在交易所进行的标准化运期交易）

# 债券评级
- 企业素质
- 经营能力
- 获利能力
- 偿债能力
- 履约情况
- 发展前景

# 债券估值
基本原理是确定现金流的贴现 “present value”
- 债券的面值和利率
- 计息间隔时间
- 嵌入式期权条款
- 税收待遇
- 其他因素
债券必要回报率 = 真实无风险收益率 + 预期通货膨胀率 + 风险溢价


# 证券投资基金概述
投资基金是指向投资者募集资金，形成独立基金财产，由专业投资机构（基金管理人）进行基金投资和管理，由基金托管人进行托管。投资人共享收益，共担风险的一种集合投资方式（信托关系）
美国：共同基金
英国/香港：单位信托基金
欧洲：集合投资基金
日本/台湾：证券投资信托基金
eg. 余额宝是一个蚂蚁金服旗下的一个基金
- 股票基金：基金以股票为主要投资对象
- 债券基金：基金以债券为主要投资对象
- 混合型基金：基金同时投资于股票和债券
- 货币市场基金：仅以货币市场工具为投资对象，期限一般在一年内，包括银行短期存款，国库券，...

## 公募基金
面向社会公众公开发售基金份额，宣传推广，募集基金而设立的基金，对象不固定，适合中小投资者
## 私募基金
是向特定合格投资者发售基金份额，募集资金而设立的基金。投资金额大，风险较大，监管机构对投资者资格和人数有限制(不超过200人)，非公开发售宣传
- 基金管理人要向中国证券投资基金业协会登记手续
- 不得公开宣称讲座报告
- 私募证券投资基金：主要投资二级市场的股票，债券，期货等上市交易品种（投资周期较短）
- 私募股权投资基金：主要投资非上市公司的股权，或者上市公司的非公开交易股权（投资周期较长，收益更高）

## ETF交易所交易基金（上市开放式基金）
是一种在交易所上市交易。基金份额可变的一种基金运作方式

# 基金如何运作管理
基金运作包括基金募集，基金投资管理，基金资产托管，基金份额的登记和交易，基金资产的估值和会计核算，基金的信息披露及其相关环节。
在基金运作中，基金管理人为主导核心地位
- 基金份额持有人（投资者），是投资的受益人，资产的所有人
- 基金管理人是基金的募集者和管理者
- 基金托管人是主要保管资产，资金清算，会计核算

# 基金资产估值
- 基金资产总值：基金全部资产的价值总和
- 基金资产净值：总值扣除所有负债
- 基金份额净值：净值除以当前基金的总份额（平均）


# 金融衍生工具
金融衍生工具又称为金融衍生产品，是和基础金融产品相对应的一个概念，指建立在基础产品或基础变量之上，其价格取决于基础金融产品价格变动的派生金融产品
- 基础变量（各类资产价格，利率，汇率，信用等级，自然现象，人类行为，...）
- 跨期性，杠杆性，联动性，不确定/高风险性
- 独立衍生工具 or 嵌入式衍生工具
